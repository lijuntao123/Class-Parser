# Class-Parse
最近学习了java class文件的结构，同时又在学习scala,所以前几日萌生了使用scala 来实现javaclass文件解析。
开始开了class文件结构的资料之后，觉得自己看明白了，学习scala的时候也是一样，把它当做java的一种变体,用了
两天时间过了一遍scala的基本语法，然后手开始痒痒，想写点代码，然后就有了这个demo。
